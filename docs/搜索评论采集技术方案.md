## 一、需求分析

| 需求项 | 说明 |
|--------|------|
| **搜索关键词** | "51talk老带新" |
| **排序方式** | 按时间最近倒排（最新优先） |

## 三、核心模块设计

### 3.1 搜索模块

**API端点**：`POST /api/sns/web/v1/search/notes`

**关键参数**：
```python
{
    "keyword": "51talk老带新",      # 搜索关键词
    "page": 1,                      # 当前页码
    "page_size": 20,                # 每页数量（建议20）
    "sort": "time_descending",      # 排序方式：最新优先
    "note_type": 0,                 # 0不限 1视频 2图文
    "filters": [...]                # 筛选条件
}
```

**排序方式映射**（`sort_type_choice`）：
| 值 | 含义 | API参数 |
|----|------|---------|
| 0 | 综合排序 | `general` |
| 1 | **最新**（选用） | `time_descending` |
| 2 | 最多点赞 | `popularity_descending` |
| 3 | 最多评论 | `comment_descending` |
| 4 | 最多收藏 | `collect_descending` |

### 3.2 评论采集模块

**一级评论 API**：`GET /api/sns/web/v2/comment/page`
**二级评论 API**：`GET /api/sns/web/v2/comment/sub/page`

**采集策略**：
```
笔记URL
  │
  ├──▶ 获取所有一级评论（分页）
  │       └── 每条一级评论
  │               └──▶ 获取该评论的所有二级回复（分页）
  │
  └──▶ 合并所有评论数据
```

**评论数据结构**：
```python
{
    'note_id': '笔记ID',
    'note_url': '笔记链接',
    'comment_id': '评论ID',
    'parent_comment_id': '父评论ID（一级为空）',
    'comment_level': 1/2,  # 1=一级评论, 2=二级回复
    'user_id': '用户ID',
    'nickname': '昵称',
    'content': '评论内容',
    'like_count': '点赞数',
    'upload_time': '发布时间',
    'ip_location': 'IP归属地',
}
```

```python
        self.engine = create_engine(
            db_url,
            pool_size=5,              # 连接池大小
            max_overflow=10,          # 最大溢出连接

            echo=False                # 是否打印SQL语句

        """获取数据库会话"""

        """关闭数据库连接"""
```

## 五、定时任务设计

### 5.1 调度配置

**使用 APScheduler 实现每小时执行**：

```python
from apscheduler.schedulers.background import BackgroundScheduler
from apscheduler.triggers.cron import CronTrigger

class CommentCrawlerScheduler:
    """评论采集定时调度器"""
    
    def __init__(self, spider):
        self.scheduler = BackgroundScheduler()
        self.spider = spider
        
    def start(self):
        """启动定时任务，每小时执行一次"""
        trigger = CronTrigger(minute='0')
        self.scheduler.add_job(
            self.run_crawl_job,
            trigger=trigger,
            id='comment_crawler_job',
            replace_existing=True
        )
        self.scheduler.start()
        
    def run_crawl_job(self):
        """执行采集任务"""
        try:
            logger.info("=== 定时任务开始执行 ===")
            self.spider.run_incremental_crawl()
            logger.info("=== 定时任务执行完成 ===")
        except Exception as e:
            logger.error(f"定时任务执行失败: {e}")
            
    def shutdown(self):
        """关闭调度器"""
        self.scheduler.shutdown()
```

```python
import hashlib
from datetime import datetime
from sqlalchemy import func
from models import DatabaseManager, CommentHistory, NoteHistory, CrawlLog

class IncrementalCommentSpider:
    """增量评论采集爬虫（SQLAlchemy版本）"""
    
    def __init__(self, db_url):
        """
        :param db_url: 数据库连接URL
        注意：需先手动执行 SQL 初始化脚本创建表
        """
        self.db_manager = DatabaseManager(db_url)
        
    def generate_comment_unique_id(self, comment):
        """生成评论唯一标识"""
        note_id = comment.get('note_id', '')
        comment_id = comment.get('comment_id', '')
        user_id = comment.get('user_id', '')
        content = comment.get('content', '')[:50]
        upload_time = comment.get('upload_time', '')
        
        unique_str = f"{note_id}_{comment_id}_{user_id}_{content}_{upload_time}"
        return hashlib.md5(unique_str.encode('utf-8')).hexdigest()
    
    def is_comment_exists(self, session, unique_id):
        """检查评论是否已存在"""
        return session.query(CommentHistory).filter_by(unique_id=unique_id).first() is not None
    
    def batch_check_exists(self, session, unique_ids):
        """批量检查评论是否存在，返回已存在的ID集合"""
        if not unique_ids:
            return set()
        
        results = session.query(CommentHistory.unique_id).filter(
            CommentHistory.unique_id.in_(unique_ids)
        ).all()
        return set(row[0] for row in results)
    
    def filter_new_comments(self, session, comments):
        """过滤出新增的评论"""
        for comment in comments:
            comment['_unique_id'] = self.generate_comment_unique_id(comment)
        
        unique_ids = [c['_unique_id'] for c in comments]
        exists_set = self.batch_check_exists(session, unique_ids)
        
        new_comments = [c for c in comments if c['_unique_id'] not in exists_set]
        return new_comments
    
    def save_comment_history_batch(self, session, comments):
        """批量保存评论历史记录"""
        if not comments:
            return 0
        
        comment_objects = []
        for comment in comments:
            ch = CommentHistory(
                unique_id=comment.get('_unique_id'),
                note_id=comment.get('note_id'),
                comment_id=comment.get('comment_id'),
                parent_comment_id=comment.get('parent_comment_id'),
                user_id=comment.get('user_id'),
                nickname=comment.get('nickname'),
                content=comment.get('content'),
                like_count=comment.get('like_count', 0),
                upload_time=comment.get('upload_time'),
                ip_location=comment.get('ip_location')
            )
            comment_objects.append(ch)
        
        session.bulk_save_objects(comment_objects)
        return len(comment_objects)
    
    def save_note_history(self, session, notes):
        """保存笔记历史记录（存在则更新）"""
        for note in notes:
            existing = session.query(NoteHistory).filter_by(
                note_id=note.get('note_id')
            ).first()
            
            if existing:
                # 更新现有记录
                existing.note_url = note.get('note_url')
                existing.title = note.get('title')
                existing.nickname = note.get('nickname')
                existing.liked_count = note.get('liked_count', 0)
                existing.collected_count = note.get('collected_count', 0)
                existing.comment_count = note.get('comment_count', 0)
                existing.share_count = note.get('share_count', 0)
            else:
                # 插入新记录
                new_note = NoteHistory(
                    note_id=note.get('note_id'),
                    note_url=note.get('note_url'),
                    title=note.get('title'),
                    user_id=note.get('user_id'),
                    nickname=note.get('nickname'),
                    note_type=note.get('note_type'),
                    liked_count=note.get('liked_count', 0),
                    collected_count=note.get('collected_count', 0),
                    comment_count=note.get('comment_count', 0),
                    share_count=note.get('share_count', 0),
                    upload_time=note.get('upload_time')
                )
                session.add(new_note)
    
    def run_incremental_crawl(self, keyword="51talk老带新", max_pages=5):
        """执行增量采集"""
        task_id = datetime.now().strftime('%Y%m%d_%H%M%S')
        today = datetime.now().strftime('%Y-%m-%d')
        
        session = self.db_manager.get_session()
        
        try:
            # 记录任务开始
            crawl_log = CrawlLog(task_id=task_id, keyword=keyword, status=0)
            session.add(crawl_log)
            session.commit()
            
            all_new_comments = []
            all_notes = []
            total_comments = 0
            
            # 1. 分页搜索笔记
            for page in range(1, max_pages + 1):
                notes = self.search_notes(keyword, page, sort_type=1)
                if not notes:
                    break
                
                all_notes.extend(notes)
                
                # 2. 采集每条笔记的评论
                for note in notes:
                    note_url = f"https://www.xiaohongshu.com/explore/{note['id']}"
                    
                    # 获取评论
                    comments = self.get_all_comments(note_url)
                    total_comments += len(comments)
                    
                    # 3. 过滤新增评论
                    new_comments = self.filter_new_comments(session, comments)
                    
                    if new_comments:
                        # 关联笔记信息
                        for comment in new_comments:
                            comment['note_id'] = note['id']
                            comment['note_url'] = note_url
                        
                        all_new_comments.extend(new_comments)
                        
                        # 4. 批量保存到MySQL
                        self.save_comment_history_batch(session, new_comments)
                        session.commit()
                    
                    # 随机延迟
                    self.random_delay(0.5, 1.5)
                
                self.random_delay(1, 2)
            
            # 5. 保存笔记历史
            self.save_note_history(session, all_notes)
            
            # 6. 保存当日增量Excel
            if all_new_comments:
                self.save_increment_data(all_notes, all_new_comments, today)
                logger.info(f"本次新增评论: {len(all_new_comments)} 条")
            
            # 7. 更新任务日志
            crawl_log.status = 1
            crawl_log.end_time = datetime.now()
            crawl_log.total_notes = len(all_notes)
            crawl_log.total_comments = total_comments
            crawl_log.new_comments = len(all_new_comments)
            session.commit()
            
            return all_notes, all_new_comments
            
        except Exception as e:
            session.rollback()
            # 更新任务失败状态
            crawl_log.status = 2
            crawl_log.end_time = datetime.now()
            crawl_log.error_msg = str(e)
            session.commit()
            raise
        finally:
            session.close()
```

### 7.1 Cookie池管理器实现

```python
from datetime import datetime
from sqlalchemy import func, case
from models import DatabaseManager, CookiePool, CookieUsageLog

class CookiePoolManager:
    """Cookie池管理器（SQLAlchemy版本）"""
    
    def __init__(self, db_manager, max_fail_count=5):
        """
        :param db_manager: DatabaseManager实例
        :param max_fail_count: 单Cookie最大失败次数
        """
        self.db_manager = db_manager
        self.max_fail_count = max_fail_count
        self.current_cookie = None
        self.current_cookie_name = None
        
    def add_cookie(self, cookie_name, cookie_value, account_info=''):
        """添加新Cookie到池中"""
        session = self.db_manager.get_session()
        try:
            # 查询是否已存在
            existing = session.query(CookiePool).filter_by(cookie_name=cookie_name).first()
            
            if existing:
                # 更新现有Cookie
                existing.cookie_value = cookie_value
                existing.account_info = account_info
                existing.status = 1
                existing.fail_count = 0
            else:
                # 创建新Cookie记录
                new_cookie = CookiePool(
                    cookie_name=cookie_name,
                    cookie_value=cookie_value,
                    account_info=account_info,
                    status=1
                )
                session.add(new_cookie)
            
            session.commit()
            logger.info(f"Cookie [{cookie_name}] 已添加到池中")
        except Exception as e:
            session.rollback()
            logger.error(f"添加Cookie失败: {e}")
        finally:
            session.close()
    
    def get_available_cookie(self, strategy='round_robin'):
        """
        获取可用的Cookie
        :param strategy: 选择策略 round_robin/random/least_used
        """
        session = self.db_manager.get_session()
        try:
            query = session.query(CookiePool).filter_by(status=1)
            
            if strategy == 'round_robin':
                # 按最后使用时间排序，最早使用的优先
                query = query.order_by(CookiePool.last_used_time.asc().nulls_first())
            elif strategy == 'random':
                # 随机排序
                query = query.order_by(func.rand())
            elif strategy == 'least_used':
                # 使用次数最少的优先
                query = query.order_by(CookiePool.use_count.asc())
            
            result = query.first()
            
            if result:
                self.current_cookie_name = result.cookie_name
                self.current_cookie = result.cookie_value
                
                # 更新使用时间和次数
                result.last_used_time = datetime.now()
                result.use_count += 1
                session.commit()
                
                return result.cookie_name, result.cookie_value
            else:
                logger.error("没有可用的Cookie，请检查Cookie池")
                return None, None
        finally:
            session.close()
    
    def report_cookie_result(self, cookie_name, is_success, error_msg=None):
        """报告Cookie使用结果"""
        session = self.db_manager.get_session()
        try:
            cookie = session.query(CookiePool).filter_by(cookie_name=cookie_name).first()
            
            if cookie:
                if is_success:
                    cookie.success_count += 1
                    cookie.last_success_time = datetime.now()
                else:
                    cookie.fail_count += 1
                    
                    # 检查是否超过最大失败次数
                    if cookie.fail_count >= self.max_fail_count:
                        cookie.status = 0
                        logger.warning(f"Cookie [{cookie_name}] 连续失败{self.max_fail_count}次，已标记为失效")
                
                session.commit()
            
            # 记录使用日志
            usage_log = CookieUsageLog(
                cookie_name=cookie_name,
                is_success=1 if is_success else 0,
                error_msg=error_msg
            )
            session.add(usage_log)
            session.commit()
            
        except Exception as e:
            session.rollback()
            logger.error(f"报告Cookie结果失败: {e}")
        finally:
            session.close()
    
    def get_pool_status(self):
        """获取Cookie池状态统计"""
        session = self.db_manager.get_session()
        try:
            results = session.query(
                CookiePool.status,
                func.count().label('count'),
                func.sum(CookiePool.use_count).label('total_use'),
                func.sum(CookiePool.success_count).label('total_success'),
                func.sum(CookiePool.fail_count).label('total_fail')
            ).group_by(CookiePool.status).all()
            
            return results
        finally:
            session.close()
    
    def print_pool_status(self):
        """打印Cookie池状态"""
        status = self.get_pool_status()
        
        print("=" * 60)
        print("Cookie池状态")
        print("=" * 60)
        
        status_map = {0: '失效', 1: '正常', 2: '冻结'}
        
        for item in status:
            status_name = status_map.get(item.status, '未知')
            print(f"状态 [{status_name}]: {item.count} 个")
            print(f"  - 总使用次数: {item.total_use or 0}")
            print(f"  - 成功: {item.total_success or 0} | 失败: {item.total_fail or 0}")
            
            if item.total_use and item.total_use > 0:
                success_rate = (item.total_success or 0) / item.total_use * 100
                print(f"  - 成功率: {success_rate:.1f}%")
        print("=" * 60)
```

### 7.2 Cookie池装饰器

```python
class CookiePoolDecorator:
    """Cookie池装饰器 - 为API请求自动管理Cookie"""
    
    def __init__(self, cookie_pool_manager):
        self.cookie_manager = cookie_pool_manager
    
    def __call__(self, func):
        def wrapper(*args, **kwargs):
            max_retry = 3
            for i in range(max_retry):
                # 获取可用Cookie
                cookie_name, cookie = self.cookie_manager.get_available_cookie()
                if not cookie:
                    raise Exception("没有可用的Cookie")
                
                try:
                    # 执行请求
                    result = func(*args, cookies_str=cookie, **kwargs)
                    
                    # 检查结果是否成功
                    if isinstance(result, tuple) and len(result) >= 2:
                        success = result[0]
                        msg = result[1] if len(result) > 1 else ''
                        
                        # 检查是否Cookie失效的特征
                        if not success and self._is_cookie_invalid(msg):
                            self.cookie_manager.report_cookie_result(cookie_name, False, msg)
                            logger.warning(f"Cookie失效，自动切换 [{cookie_name}] -> 下一个")
                            continue
                        else:
                            self.cookie_manager.report_cookie_result(cookie_name, success)
                            return result
                    else:
                        self.cookie_manager.report_cookie_result(cookie_name, True)
                        return result
                        
                except Exception as e:
                    error_msg = str(e)
                    self.cookie_manager.report_cookie_result(cookie_name, False, error_msg)
                    
                    # 判断是否需要切换Cookie重试
                    if self._is_cookie_invalid(error_msg) and i < max_retry - 1:
                        logger.warning(f"请求异常，切换Cookie重试 [{i+1}/{max_retry}]")
                        continue
                    else:
                        raise
            
            raise Exception(f"重试{max_retry}次后仍然失败")
        
        return wrapper
    
    def _is_cookie_invalid(self, error_msg):
        """判断错误是否由Cookie失效导致"""
        invalid_keywords = [
            '登录', 'cookie', 'session', '失效', '过期',
            '未登录', '鉴权', 'auth', 'forbidden', 'unauthorized'
        ]
        error_lower = str(error_msg).lower()
        return any(keyword in error_lower for keyword in invalid_keywords)
```

## 十、异常处理与限流策略

### 10.1 异常分类处理

| 异常类型 | 处理策略 |
|----------|----------|
| **请求失败**（网络/超时） | 使用 `@retry` 装饰器，3次重试，指数退避 |
| **签名失效**（X-s验证失败） | 检查Cookie有效性，提示重新登录 |
| **频率限制**（429/滑动验证） | 随机延迟 1-3 秒，增加代理池轮换 |
| **数据缺失**（字段为空） | 使用默认值填充，记录日志 |
| **SQLAlchemy连接失败** | 连接池重试，最多5次 |
| **事务执行失败** | 自动回滚，记录错误日志 |

```python
from contextlib import contextmanager

@contextmanager
def get_db_session(db_manager):
    """数据库会话上下文管理器"""
    session = db_manager.get_session()
    try:
        yield session
        session.commit()
    except Exception as e:
        session.rollback()
        raise e
    finally:
        session.close()
```

## 十一、风险与应对

| 风险点 | 应对措施 |
|--------|----------|
| **Cookie失效** | **Cookie池机制**：多账号轮换，自动失效检测与切换 |
| **单点故障** | 至少配置3-5个可用Cookie，避免单Cookie失效导致系统停摆 |
| **频率限制** | 多Cookie负载均衡，分散请求压力 |
| **账号封禁** | Cookie池健康检查，自动剔除异常账号 |
| 反爬升级 | 监控请求成功率，及时调整延迟策略 |
| 数据量大 | SQLAlchemy连接池+批量操作优化 |
| 评论数据过多 | 设置单条笔记最大评论数限制 |
| 定时任务重叠 | 使用数据库锁或文件锁防止重复执行 |
| MySQL连接耗尽 | SQLAlchemy连接池自动管理 |
| 存储空间不足 | 定期归档历史数据，分区表管理 |

### 11.3 Cookie池初始化

```python
# init_cookie_pool.py
# 注意：运行前需先手动执行 init.sql 创建数据库表
from config import DB_CONFIG, COOKIE_POOL
from models import DatabaseManager
from cookie_pool_manager import CookiePoolManager

def init_cookie_pool():
    # 确保已手动执行 SQL 脚本创建表
    db_manager = DatabaseManager(DB_CONFIG['url'])
    cookie_manager = CookiePoolManager(db_manager)
    
    for cookie_info in COOKIE_POOL:
        cookie_manager.add_cookie(
            cookie_info['name'],
            cookie_info['value'],
            cookie_info['account_info']
        )
    
    print("Cookie池初始化完成")
    cookie_manager.print_pool_status()

if __name__ == '__main__':
    init_cookie_pool()
```
